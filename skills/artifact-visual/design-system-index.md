# Design System Index

> Source: [getdesign.md](https://getdesign.md/) — 66 curated DESIGN.md files
> extracted from real brand websites. Use this index to select the best-matching
> design system for a user's website generation request.

## How to Use This Index

1. Read the user's query and any scraped content signals (industry, tone, audience)
2. Find the brand whose **Style** and **Best For** best match the request
3. Run `npx getdesign@latest add <id>` to download the DESIGN.md
4. If the user explicitly requests a visual style (e.g., "dark premium", "warm and
   friendly"), prioritize that over industry matching
5. If no brand is a reasonable match, skip — the artifact-visual skill will provide
   a default design system
